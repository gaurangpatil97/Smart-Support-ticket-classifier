Key findings
🔍

Dirty data inflated metrics

EOL got 1.00 F1 on 44 identical templates. The audit was more valuable than any model.

🧠

Embeddings beat TF-IDF

They added +0.05 macro F1 and +0.09 on real tickets. Semantic meaning matters more than word frequency on short noisy text.

⚡

GPT wins on clean prompts only

It reached 0.86 on the custom set, but only 0.68 on real helpdesk traffic. Prompt design doesn't fix distribution shift.

🔬

Synthetic data shifted the optimization target

Real-ticket F1 dropped 0.83 → 0.73. LLM-generated tickets are too clean to represent real noise.
